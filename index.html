About


Weathery is a Weather App developed as a Flutter project by AryanshDev which tends to provide a more personalized experience to users and provide them with only those values that matter to them, discarding all other lesser useful information.

Weathery also provides weather alerts to help users safegaurd themself from upcoming dangers.





Clear The Clutter

Weathery Provides Only Those Values That Matter, Discarding Useless Values That Don't Matter To Normal People



Stay Alert, Always

Weathery Provides Alerts Issued In Your Area So You Know About Upcoming Dangers And Plan Accordingly



See World Weather

Weathery Allows Users To Easily Get Weather Details of Anyplace Around The Globe Anytime.
